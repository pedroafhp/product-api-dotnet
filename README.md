# API REST em C# (.NET)

Este projeto é uma API REST desenvolvida em C# utilizando ASP.NET Core Web API, com foco em aprendizado e construção de portfólio backend.

A API implementa um CRUD completo de produtos, seguindo boas práticas de organização e estruturação de projetos.

---

# Objetivo

Este projeto tem como objetivo demonstrar conhecimentos em:

- Desenvolvimento de APIs REST com C#
- Estruturação de projetos backend
- Operações CRUD (Create, Read, Update, Delete)
- Uso de Entity Framework Core
- Integração com banco de dados
- Documentação com Swagger

---

# Tecnologias Utilizadas

- C#
- .NET (6/7/8)
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server (ou banco utilizado)
- Swagger

---

# Funcionalidades

## Produtos (CRUD)

- Criar produto
- Listar todos os produtos
- Buscar produto por ID
- Atualizar produto
- Remover produto
